## Comandos Úteis

```bash

docker ps -a
docker-compose up -d
docker logs -f container

# Git
git status
git diff
git log --oneline -10

# Node
npm run dev
npm run build

# Python
python -m venv venv
source venv/bin/activate
python main.py
```
